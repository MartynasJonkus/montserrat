To run the react frontend  
  cd \client\  
  npm install  
  npm run dev  

To run ASP.NET api
  dotnet run

Or inspect API with swagger
  dotnet watch run

## Installations
- Visual Studio Code
- Node.js
- .NET SDK 8.0
